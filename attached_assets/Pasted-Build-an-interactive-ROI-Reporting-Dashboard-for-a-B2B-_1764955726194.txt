## What to Build
A responsive web dashboard with 3 views:
1. Executive Summary (landing page)
2. Influenced Hires Detail (data table)
3. Adoption Metrics (charts)

## Tech Stack
- React + TypeScript
- Tailwind CSS for styling
- Recharts or Chart.js for visualizations
- Use a modern, professional design (think Tableau/Looker aesthetic)
